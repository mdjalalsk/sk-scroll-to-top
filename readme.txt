This plugin is lightweight, customizable, and easy to integrate into any WordPress website.

== Description ==

The "SK Scroll to Top" plugin adds a smooth, customizable Scroll to Top button for a better user experience on your WordPress site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sk-scroll-to-top/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, the Scroll to Top button will appear on your website, allowing users to smoothly scroll to the top.
